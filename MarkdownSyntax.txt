Заголовки
Для создания заголовков используются символы #. Количество символов # указывает уровень заголовка. Например:
# Заголовок 1
## Заголовок 2
### Заголовок 3


Параграфы и переносы строк
Для создания параграфов достаточно оставить пустую строку между текстовыми блоками. Для переноса строки внутри параграфа можно использовать два пробела в конце строки.
Это первый параграф.
Это второй параграф.  
Следующая строка второго параграфа.


Списки
Markdown поддерживает создание упорядоченных (нумерованных) и неупорядоченных (маркированных) списков.
Упорядоченный список создается с помощью чисел или точек:
1. Пункт 1
2. Пункт 2
3. Пункт 3
Неупорядоченный список создается с помощью -, * или +:
- Пункт 1
- Пункт 2
- Пункт 3


Ссылки
Ссылки в Markdown могут быть добавлены с помощью следующего синтаксиса:
[Текст ссылки](URL)


Изображения
Изображения в Markdown можно вставить с помощью следующего синтаксиса:
![Альтернативный текст](URL_изображения)


Код
Чтобы вставить блок кода, используйте тройные обратные кавычки (```) перед и после кода. Можно также указать язык программирования для правильной подсветки синтаксиса.
Пример:
<pre>
```python
print("Hello, World!")
```
</pre>


Дополнительные возможности
Markdown-синтаксис включает множество других возможностей, таких как:
Жирный текст: **текст** или __текст__
Курсив: *текст* или `_текст_`
Зачеркнутый текст: ~~текст~~
Список задач:
- [x] Выполнить задачу 1
- [ ] Выполнить задачу 2
- [ ] Выполнить задачу 3
Цитаты:
> Это цитата.
